The following are some of the labs used in CS482 from Fall 2016 to Spring 2017. Many of them were refined from the SEED Project. The type of operating system used for this lab is noted within the brackets. Please address any questions to usma@benklim.org. Also, you may create an issue on the CS482 git repo to have your edits or suggestions incorporated.
  • Linux Hardening Lab [Ubuntu 16.04]
  • Windows Hardening Lab [Windows 7]
  • TCP/IP Attack Hardening Lab [3 x Ubuntu 16.04]
  • DNS Configuration Lab [2 x Ubuntu 16.04, Windows Server 2008]
  • Kaminsky Attack Lab [3 x Ubuntu 12.04]
  • Firewall Lab [3 x Ubuntu 16.04]
  • Web Apps Lab [2 x Ubuntu 16.04]
  • SQLi/XSS Lab [3 x Ubuntu 16.04]
  • PKI Lab [3 x Ubuntu 16.04]
  • rtn2libc Lab [Ubuntu 16.04]
  • Shellshock Lab [Ubuntu 12.04, Ubuntu 16.04]
  • Incident Handling Lab [Ubuntu 16.04]
